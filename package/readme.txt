To run the game with dance pad:
1. Connect your dance pad
2. Launch run.bat
3. Switch focus back from GlovePIE to the command window of run.bat executed in step 2
4. Press any key
5. Enjoy the ride!


The game runs in 1024x768 fullscreen resolution.


Thank you for playing!
